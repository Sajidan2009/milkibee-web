## 🍯 Tentang Milkibee

Milkibee adalah minuman susu segar berpadu madu alami yang hadir dalam 4 varian rasa:
- 🍯 **Original Honey** — Klasik manis alami
- 🍫 **Chocolate Honey** — Coklat premium + madu
- 🌿 **Mint Honey** — Segar menyegarkan + madu
- 🌼 **Vanilla Honey** — Vanilla lembut + madu

## 🚀 Cara Menjalankan

Website ini adalah **static HTML** — tidak memerlukan server atau build tools.

### Option 1: Buka Langsung
```bash
# Clone repo
git clone https://github.com/username/milkibee.git
cd milkibee

# Buka di browser
open index.html
```

### Option 2: Local Server (Recommended)
```bash
# Pakai Python
python3 -m http.server 8080

# Atau pakai Node.js (npx)
npx serve .

# Buka browser: http://localhost:8080
```

## 🌐 Deploy ke GitHub Pages

1. Push semua file ke GitHub repository
2. Buka **Settings** → **Pages**
3. Source: **Deploy from a branch**
4. Branch: **main** / root `/`
5. Klik **Save**
6. Website live di: `https://username.github.io/milkibee`
